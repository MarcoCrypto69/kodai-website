## Projet KodAI — Système IA email

### Contexte
- Réservations via courriel (pas de système en ligne)
- Clients: Québécois, Canadiens, Américains, **Français** (numéro gratuit FR actif)
- Volume estimé: élevé en haute saison (hiver + été)
- Site bien fait mais pas d'agent IA

### Solution proposée
1. **Agent IA email** — lit les courriels entrants, rédige brouillon dans la langue du client
2. **Widget chat site** — répond aux questions 24/7 avant même l'email
3. Knowledge base = ce fichier + disponibilités via Google Sheets

### ROI pour Sébastien
- Forfait min = $1,095/pers × 2 = $2,190/réservation
- Fee mensuel proposé: $250-400/mois
- 1 réservation supplémentaire/mois = payé 5-8× over

### Pricing proposé
| Service | Setup | Mensuel |
|---------|-------|---------|
| Email IA seulement | $2,500 | $300/mois |
| Chat site + email IA | $4,000 | $500/mois |

### Stack technique
- Gmail API (lecture + création brouillons)
- Claude API Haiku (pas cher, rapide)
